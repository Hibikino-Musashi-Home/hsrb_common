Overview
+++++++++

HSR-Bのロボット記述ファイルなど基本構成に関わるパッケージ群。

パッケージ一覧
++++++++++++++

- **hsrb_description**  ロボットモデル（URDF、Gazebo用記述）
- **hsrb_common_config** コントローラ・センサなどの実機・Gazebo共通設定ファイル管理パッケージ
- **hsrb_parts_description**  ロボットモデルのセンサ類パーツ（URDF、Gazebo用記述）
- **hsrc_description**  HSR-C用ロボットモデル（URDF、Gazebo用記述）
